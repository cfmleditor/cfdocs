# cftreeitem

Populates a form tree control, created with the cftree tag,
 with elements. To display icons, you can use the img values
 that CFML provides, or reference your own icons.

### Syntax

```html
<cftreeitem>
```

### Script Syntax ACF11+, Lucee, Railo 4.2+

```javascript
cftreeitem();
```
