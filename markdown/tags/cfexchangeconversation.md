# cfexchangeconversation

Helps users organize and manage conversations from a Microsoft Exchange account.

### Syntax

```html
<cfexchangeconversation action="" connection="" folderID="" name="">
```

### Script Syntax ACF11+, Lucee, Railo 4.2+

```javascript
cfexchangeconversation(action="", connection="", folderID="", name="");
```
