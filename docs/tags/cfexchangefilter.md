# cfexchangefilter

Specifies the filter parameter for cfexchangemail, cfexchangecalendar, cfexchangetask, and cfexchangecontact, get operations.

### Syntax

```html
<cfexchangefilter name="allDayEvent">
```

### Script Syntax ACF11+, Lucee, Railo 4.2+

```javascript
cfexchangefilter(name="allDayEvent");
```
