# cfprocparam

Defines stored procedure parameters.
This tag is nested within a `cfstoredproc` tag.
This tag does not have a body.

### Syntax

```html
<cfprocparam cfsqltype="CF_SQL_BIGINT">
```

### Script Syntax ACF11+, Lucee, Railo 4.2+

```javascript
cfprocparam(cfsqltype="CF_SQL_BIGINT");
```
