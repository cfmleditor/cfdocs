# cf_socialplugin

Dynamically generate social elements for Facebook, Google +, Twitter.

The supported widgets are:

Like button
Tweet button
Facebook comment box
Google Plus button
Facebook subscribe button
Like box
Activity feed
Follow

### Syntax

```html
<cf_socialplugin .. />
```

### Script Syntax ACF11+, Lucee, Railo 4.2+

```javascript
cf_socialplugin(.. /);
```
