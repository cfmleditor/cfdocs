# cfloop

Different items are required based on loop type. Items listed
 as required may not be depending on your loop type.
 Loop forms:
 [query] [condition] [index + from + to ] [index + list]
 [collection + item ]

### Syntax

```html
<cfloop>
```

### Script Syntax

```javascript
for (i = 1; i <= 10; i++) { }
```
