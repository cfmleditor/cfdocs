# cfsharepoint

Invokes a feature that SharePoint exposes as a web service action, such as the Document Workspace
getdwsdata action.

### Syntax

```html
<cfsharepoint action="cancreatedwsurl">
```

### Script Syntax ACF11+, Lucee, Railo 4.2+

```javascript
cfsharepoint(action="cancreatedwsurl");
```
