# cfexchangecontact

Creates, deletes, modifies, and gets Microsoft Exchange contact records, and gets contact record attachments.

### Syntax

```html
<cfexchangecontact action="create" contact="" name="" uid="">
```

### Script Syntax ACF11+, Lucee, Railo 4.2+

```javascript
cfexchangecontact(action="create", contact="", name="", uid="");
```
